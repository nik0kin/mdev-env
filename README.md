# mdev-env
mule development environment

### Dependencys
 see http://github.com/nik0kin/mule for dependencys

### Install

```
npm install
./installGames.sh
```

### Run

```
node server.js
# or
npm start
```

### Develop

Pull newest versions of game code from git:
```./updateGames.sh```


Run this once, to be able to commit changed mule-core code:
```./prepareDev.sh```
